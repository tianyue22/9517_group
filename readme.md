## 加分
鲁棒性实验:在测试图像加入失真并重新评估。
失真类型至少做几种:噪声、模糊、低亮度/低对比度、部分遮挡。
少样本实验:减少训练数据(如75%、50%)看性能变化。
标注质量实验:模拟较差标注或讨论弱标注影响(可做小规模)。
改进并验证:基于上面分析提出改进，再做前后对比。
可选改进方向:模型结构调整、loss修改、训练策略(增强/采样/学习率)或后处理。
失败分析:明确"失败类型-原因-对应改进"的证据链。
